How to connect to Pace University WiFi:

1. Select “PACE-WIFI” from the list of networks
2. Select “Protected EAP (PEAP)” for authentication
3. Check the “No CA certificate required” box
4. These should already be set by default, but if not, the PEAP version should be “Automatic”,
the Inner Authentication field should be “MSCHAPv2”, the Security field should be “WPA2 & WPA2 Enterprise”,
and the Anonymous Identity and Domain fields can be left blank
5. Enter your username and password (your username is your email without @pace.edu)
6. Click Connect
